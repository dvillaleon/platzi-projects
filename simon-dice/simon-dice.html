JavaScript Simon Says

Instructions

Game Goal

Repeat the same exact sequence from Simon.

1-player Game

Press the "Start game!" button. Simon will give the first signal. Repeat the signal by pressing the same color lens.

Simon will duplicate the first signal and add one. Repeat these two signals by pressing the same color lenses, in order.

Continue playing as long as you can repeat each sequence of signals correctly.

The game ends if you fail to repeat the same exact sequence, which means you have lost, or if you complete the entire sequence without a miss, in that case, you win the game

2 or More Player Game

Players may team up against Simon or compete with each other.

Decide who goes first. Proceed to play as described above except players take turns repeating Simon's signals.
